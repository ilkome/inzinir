Станок токарный с ЧПУ

DELTAMAB 400
МашиностроениеМеталлообработка388

Назначение
Изготовление деталей типа тел вращения

Описание
Прецизионный токарный станок с ЧПУ выполнен по прогрессивной схеме компоновки с расположением револьверной головки на наклонных направляющих, что увеличивает жесткость конструкции, позволяет уменьшить размеры станка по ширине и оптимально распределить по направляющим нагрузки, образующиеся в процессе резания.

Основные характеристики
Наибольший диаметр обрабатываемого изделия – 600 мм
Число управляемых координат – 2

Год выпуска: 2012

Производитель:SOMAB, Франция

Организация (ВУЗ / ГНЦ): Национальный исследовательский Томский политехнический университет
